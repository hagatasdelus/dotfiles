4. ビジュアルモード（範囲選択）

操作内容

キー

文字単位のビジュアルモード開始

v

行単位ビジュアルモード開始

V（Shift+v）

矩形（ブロック）選択ビジュアルモード開始

Ctrl+v



ビジュアルモード中は、h/j/k/l や w/b などで選択範囲を広げ、y でコピー、d で削除などを行います。

5. 便利なチェンジ系（中級向け）

操作内容

キー

ある「囲み」の内側を消して入力し直す

ci + 囲み記号（例: ci" でダブルクォート内だけ書き換え）

囲みごと（記号も含めて）消して入力し直す

ca + 囲み記号（例: ca" で "..." 全体を置き換え）

囲みの内側を選択

vi + 囲み記号（例: vi"）

囲みごと選択

va + 囲み記号（例: va"）

配列リテラルなど「括弧の内側」を削除

di + 括弧（例: di], di) など）
